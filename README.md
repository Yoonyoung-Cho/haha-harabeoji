# 우리들의 즐거운 시간 (우즐시)

> 매일 아침, 당신의 입가에 피어나는 작은 미소

70~90대 어르신이 외로움을 해소하고 쉽게 유머·건강·지혜 글을 즐길 수 있는 웹 앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 을 열면 됩니다.

## 기능

- **큰 글씨·큰 버튼**: 배경 미색(#F5F0E8), 본문 24px 이상, 고대비 색상
- **스크롤 피드**: 유머 / 건강 / 지혜 카테고리별 카드 스크롤
- **좋아요(저장)**: 카드 우측 상단 하트로 저장/해제, 상단 “내가 좋아요 누른 게시글”에서 모아보기
- **평가(따봉)**: 👍/👎 버튼으로 게시글 평가 (선택 상태는 브라우저에 저장, 누르면 다음 카드로 이동)
- **공유하기**: Web Share API로 카카오톡 등 앱에 공유 (미지원 시 클립보드 복사), 카드 우측 상단에 공유 수 표시

## 기술 스택

- Next.js (App Router), TypeScript, Tailwind CSS, Lucide React
